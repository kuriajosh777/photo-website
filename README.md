# Photo Folio

This is my photography portfolio website project. It's a simple webpage that shows off a gallery of photos, a short biography about me, and a contact form for people to reach out.

## Author

Josh Kuria

## About the Project

I built this project to practice using HTML and CSS to create a portfolio-style website. The page includes:

- An intro section describing the site
- A gallery of favourite photos
- A biography section with a picture and some info about me
- A contact form so visitors can send a message

The whole page is styled using a custom stylesheet (`style.css`) that I wrote myself. I used things like selectors, borders, background colours, font styling, and hover effects to make the page look neat without making the CSS too complicated.

## Project Setup

To run this project on your own computer:

1. Clone this repository
   ```
   git clone <repo-url>
   ```
2. Open the project folder
   ```
   cd photo-folio
   ```
3. Open `index.html` in your browser (you can just double-click the file, or right-click it and choose "Open with" your browser)

No installation or extra tools are needed since this is just plain HTML and CSS.
